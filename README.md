# Chrono Soul Archive

## Описание
`Chrono Soul Archive` — MVP Telegram Mini App в псевдонаучно-эзотерическом стиле. Пользователь вводит дату рождения, город и имя (опционально), после чего получает последовательность интерпретаций «прошлых жизней».

Проект собран как быстрый прототип для проверки виральной механики:
- Жизнь #1 — бесплатно
- Жизнь #2 — бесплатно
- Жизнь #3 — открывается за репост/шэр
- Жизнь #4 и далее — за Stars (в текущем MVP реализовано как демо-кнопка, без backend-платежей)

## Фичи
- Полноценный главный экран-лендинг: заголовок, hero-изображение и кнопки `Learn`/`About the project`.
- Отдельная страница `Learn` с рабочим сценарием анализа прошлых жизней.
- Внутренняя страница `Description` с описанием концепции.
- Длинный стилизованный дисклеймер + обязательное подтверждение перед стартом.
- Форма ввода входных данных для реконструкции.
- Генератор жизней на базе шаблонов в `data/` с вариативными результатами.
- Выдача результатов по шагам с нужной монетизационной логикой.
- Кнопка шэра для разблокировки 3-й жизни.
- Демо-механика разблокировки 4+ жизней через Stars.
- Сохранение прогресса в `localStorage`.

## Структура
- `index.html` — главный лендинг-экран
- `learn.html` — страница запуска анализа и выдачи жизней
- `description.html` — внутренняя страница с описанием
- `REQUIREMENTS.md` — продуктовые и технические требования по неизменности результатов
- `styles.css` — стили интерфейса
- `app.js` — основной клиентский сценарий
- `assets/images/hero-placeholder.svg` — плейсхолдер-изображение для главного экрана
- `data/constants.js` — текстовые и контентные константы
- `data/generator.js` — генерация карточек прошлых жизней

## Как запустить локально
1. Открой папку проекта `projects/chrono-soul-archive`.
2. Запусти любой локальный статический сервер (например, Live Server в Cursor/VSCode).
3. Открой `index.html` в браузере и пройди сценарий MVP.

## Что подключить следующим шагом
- Backend API для верификации шэра.
- Реальные платежи Telegram Stars и webhook-подтверждения.
- Аналитику событий для воронки и конверсий.
